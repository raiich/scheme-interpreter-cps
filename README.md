# scheme-interpreter-cps

## Install & Getting Started

Depend on Gauche.

```
$ brew install gauche
$ git clone https://github.com/signdoubt/scheme-interpreter-cps.git
$ cd scheme-interpreter-cps
$ gosh -I. eval-cps.scm
;your input:
(+ 1 2)
;> 3
;your input:
```

  to exit: need `ctrl-c` ... (not `ctrl-d` !!)
